# Contributing

Thanks for your interest in contributing to codex-warp-docs.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b your-feature`
3. **Make your changes** and commit: `git commit -m "Description"`
4. **Push**: `git push origin your-feature`
5. **Open a Pull Request**

## Guidelines

- Keep documentation clear and concise
- Use proper Markdown formatting
- Test any included shell commands

## Issues

Report bugs, suggest improvements, or ask questions via [GitHub Issues](https://github.com/kinggkl/codex-warp-docs/issues).
